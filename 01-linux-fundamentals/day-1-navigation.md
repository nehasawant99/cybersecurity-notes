Day 1 - Linux Navigation

Today I learned basic Linux terminal commands:

pwd - present working directory

ls - files in the directory or folder

cd - change directory

mkdir - create a new directory

touch - create a new file

cat - print content inside the file

cp - copy content from a file

mv - move or rename a file

rm - remove file

I also created files and folders inside Kali Linux and practised navigating the filesystem.
