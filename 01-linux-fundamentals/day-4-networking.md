Day 4 - Networking Basics

Today I learned basic networking commands in Kali Linux:

ip a - display IP address and network interfaces

ip route - show routing table and default gateway

ping - test network connectivity to a host

nslookup - find the IP address of a domain

dig - retrieve detailed DNS information

ss -tuln - view listening TCP and UDP ports

arp -a - display ARP table and connected devices

hostname - show the system hostname

traceroute - trace the path packets take to a destination

nmap - scan a host for open ports and services

I also practised identifying my IP address, checking network connectivity, performing DNS lookups, viewing open ports, and running basic network scans in Kali Linux.

Concepts Learned

* IP Address
* MAC Address
* DNS
* Gateway
* TCP and UDP
* Ports
* Routing
* Network Scanning
